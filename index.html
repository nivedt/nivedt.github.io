Variables,expressions and statements

Hey there! This is me introducing to my first blog, here I will give you a glimpse of what variables, expressions and statements are in python.
Variable is a fundamental concept in python. It is a place where we can store values. Instead of using the values directly, we can refer to the variable name where the actual values are stored.
It is a reserved memory location that stores and manipulates data. We have different data types of variables, the rules for naming variables in Python. The values belong to different types: 25 is an integer, and “Hey World!” is a string, so-called because it contains a “string” of letters. These can be identified as strings because they are enclosed in quotation marks.

Example
> x = 2 === > here X is Variable
> name = 'nived' === > here NAME is variable

The expression contains values, variables, operators, and calls to functions. Expressions
need to be evaluated. If you ask Python to print an expression, the interpreter evaluates the
expression and displays the result.

Example > 25 > x > y = x + 25

A statement is a logical unit of code that can be executed at the Python interpreter. When we
type a statement at the command prompt, it will execute the code in the statement then
provides the results, as long as the code is clean. We can execute code that is in a file. In
general, these file contains scripts, these scripts are a combination of statements to perform
units of work.

Example
> print("Hey there!!")
